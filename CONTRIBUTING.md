# Contributing

Thanks for helping improve Zhijuan Prompt Card.

## Welcome Contributions

- bug fixes
- browser compatibility
- prompt schema improvements
- UI polish
- visual history improvements
- generator presets
- documentation
- i18n
- model compatibility notes

## Not Accepted

- hidden telemetry
- forced cloud dependency
- hardcoded private API endpoints
- paywall / license lock inside open-source core
- unrelated framework rewrite
- large UI rewrite without issue discussion

## Pull Request Flow

1. Open an issue first for large changes.
2. Keep PRs small and focused.
3. Run `npm run typecheck`.
4. Run `npm run build`.
5. Attach screenshots for UI changes.
6. Describe privacy impact when network behavior changes.

## Commit Style

Use conventional prefixes:

- `feat:`
- `fix:`
- `docs:`
- `refactor:`
- `chore:`
- `test:`
