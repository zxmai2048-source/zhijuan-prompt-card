# Security

Zhijuan Prompt Card is a Manifest V3 Chrome extension that calls only the configured OpenAI-compatible endpoint.

API keys are stored in `chrome.storage.local`. Do not configure endpoints or keys you do not trust. For private images, use local BridgeDeck.

The extension does not include login, Supabase, Firebase, Stripe, WeChat, Google identity, telemetry, analytics, or payment code.
