# Screenshot Guide

需要这些真实截图：

1. Floating panel
2. Quick history rail
3. Popup visual history
4. Options page
5. Prompt output tabs
6. Local history card

推荐尺寸：

- README hero screenshot: 1600x900
- Floating panel: 1200x900
- Popup history: 900x1200
- Options page: 1400x900
- Social preview: 1200x630

文件命名：

- `docs/assets/readme/floating-panel.png`
- `docs/assets/readme/quick-history.png`
- `docs/assets/readme/popup-history.png`
- `docs/assets/readme/options.png`

手动步骤：

1. `npm run build`
2. 打开 `chrome://extensions`
3. Load unpacked `dist/`
4. 配置本地 endpoint
5. 识别 3-5 张测试图片
6. 打开 floating panel / history / options
7. 截图并替换 SVG 占位图
